# Validation Goals

We present the goals for which we tested our scenario (See Section 4.1). Each goal is a user-defined goal in English, which contains the desired state of the lab and the presence of a human in a zone or both.

The first goal is: “Zone 1 should have light level 1. Zone 2 light level 2. Thereis a human in zone 1.”

The second goal is: “Zone 1 should have light level 2. Zone 2 should have light level 3.”.

The third goal is: “Zone 1 should have light level 1. Zone 2 light level 2.”.

The fourth goal is: “Zone 1 should have light level 3. Zone 2 light level 3. There is a human in zone 1. There is a human in zone 2.”

Each goal, except for the fourth goal, is successfully achieved by the MAS and the user gets notified that the goal has been achieved. The fourth goal cannot be achieved due to being impossible to materialize and the user is notified that the goal has not been achieved
