# Soar Agent

A Soar agent whose purpose is to create annotations from a perceived goal annotation.

The Soar code for the agent is present [`here`](src/main/java/soar/agents/light_processing.soar).

The code that is actually run is present [`here`](src/main/java/soar/SoarWebAgent.java).

## TO RUN

On Linux and macOS:

````
./gradlew run
````

On Windows:

````
.\gradlew.bat run
````
